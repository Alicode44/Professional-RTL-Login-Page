# Professional RTL Login Page - Bootstrap 5 🔑

A lightweight, responsive, and modern login page built with **Bootstrap 5**. This project is designed specifically for Persian (RTL) layouts with a focus on user experience and clean aesthetics.

## ✨ Features
- **Bootstrap 5 Framework**: Utilizes the latest version for layout and components.
- **RTL Support**: Fully optimized for Right-to-Left languages like Persian.
- **Responsive Design**: Mobile-first approach, looks great on all devices.
- **Form Validation**: Integrated HTML5 client-side validation.
- **Modern UI**: Clean card-based design with custom button styling.
- **Custom Fonts**: Optimized for Persian typography.

## 🛠 Tech Stack
- **HTML5** & **CSS3**
- **Bootstrap 5.x** (CSS & JS)
- **Persian Web Fonts** (Vazir / Samim)

## 🚀 How to Use

### 1. Clone the repository
```bash
git clone https://github.com/your-username/bootstrap5-login-page.git
```

### 2. Include Bootstrap 5
This project uses Bootstrap 5 CDN. Make sure you have an active internet connection or download the files locally.
```html
<!-- Bootstrap RTL CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.rtl.min.css">
```

### 3. Open the Project
Simply open `index.html` in your browser to view the login form.
